User selects from predefined arguments about whether jockeys or an auto racetrack should be used.
System uses NLP to decide which option is more appropriate, based on the user's arguments.


On screen: series of predefined statements
User: submits chosen statements to backend, e.g. using check boxes
Backend: uses NLP library to generate an answer and a weight; write these to a file
On screen: write the answer and weight
